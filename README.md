LIBRARY MANAGEMENT SYSTEM
==========================

Features:
- Complete Login System (Admin + Student)
- Book Management (Add, Delete, Search, Display)
- Member Management
- Issue & Return Books
- Library Statistics
- Proper use of Classes and OOP

OOP Concepts Used:
• Encapsulation
• Composition (Library contains Books & Members)
• Friend Function
• Static Data Member & Static Function
• Arrays of Objects
